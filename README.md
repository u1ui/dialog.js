# dialog.js
alert, prompt, confirm but async and stylable

# Ussage

```html
<script type=module>
    import {alert, prompt, confirm} from 'https://jsdelivr.net/gh/u1ui/dialog.js@x.x.x/dialog.js';

    const name = await prompt('What is your name?', 'John Doe');
</script>
```
